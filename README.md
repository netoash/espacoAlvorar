# 🌱 Espaço Alvorar

Sistema institucional e plataforma de gestão para atendimento em Fisioterapia Pediátrica, Estimulação Precoce e acompanhamento do desenvolvimento infantil.

---

## 📖 Sobre o Projeto

O Espaço Alvorar nasceu com o objetivo de oferecer uma experiência digital moderna para famílias e profissionais da área da saúde infantil.

O projeto está dividido em duas grandes áreas:

### 🌐 Site Institucional

Apresentação da clínica, serviços, equipe, contato e agendamento.

### 🏥 Plataforma Administrativa

Gestão de pacientes, agenda, prontuários, evoluções clínicas, documentos e controle financeiro.
